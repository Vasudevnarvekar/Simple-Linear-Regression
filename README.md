# Simple Linear Regression – Salary Prediction

## 📌 Project Overview

This project demonstrates the implementation of **Simple Linear Regression** to predict employee salaries based on their **Years of Experience**.
The goal is to understand the relationship between experience and salary using statistical and machine learning techniques.

The project includes data exploration, visualization, model training, and evaluation using Python libraries.

---

## 📂 Dataset

The dataset used in this project contains two main features:

* **YearsExperience** – Number of years of professional experience
* **Salary** – Salary corresponding to the experience level

The dataset is loaded using **Pandas** and analyzed before building the regression model.

## ⚙️ Project Workflow

1. **Import Libraries**
   Required Python libraries for data analysis, visualization, and modeling are imported.

2. **Load Dataset**
   The salary dataset is loaded using Pandas.

3. **Exploratory Data Analysis (EDA)**

   * View dataset structure
   * Check data types
   * Generate summary statistics
   * Detect missing values

4. **Outlier Detection**
   Boxplots are used to identify potential outliers in the dataset.

5. **Data Splitting**
   The dataset is divided into **training (70%)** and **testing (30%)** sets.

6. **Model Building**
   A **Simple Linear Regression** model is created to analyze the relationship between:

   * Independent Variable: YearsExperience
   * Dependent Variable: Salary

7. **Model Evaluation**
   The model is evaluated using statistical metrics and correlation analysis.
